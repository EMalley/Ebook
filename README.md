# EBook Mastery

EBook Mastery is a simple static website built using the Bootstrap 5 framework. The project incorporates SASS and SCSS to introduce custom classes and enhance Bootstrap's default styles, resulting in a unique and customized appearance.

## Usage

EBook Mastery is a conceptual static website designed around the theme of showcasing and exploring ebooks. The website's user interface has been artistically crafted using Bootstrap 5, coupled with bespoke modifications using SASS and SCSS. The project illustrates how a platform for navigating through a collection of ebooks and accessing related information could be developed.

## Features

- User-friendly interface designed using Bootstrap 5.
- Custom SASS and SCSS styles to enhance the default Bootstrap appearance.
- Responsive design, ensuring a seamless experience across various devices.
